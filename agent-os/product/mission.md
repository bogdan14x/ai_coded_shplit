# Product Mission

## Problem

This project aims to allow users to create a page called a sheet for tracking group expenses for a particular purpose, be it a trip or something else, allows a creator to invite participants who can access the sheet via direct URL, and everyone can add expenses to the sheet and specify who paid for what item and how the split was made, equally among participants for that specific expense, or custom split. This would all be selected when adding the expense. The ultimate goal is then to display the minimum number of transactions to settle up using the Fork Fulkerson algorithm.

## Target Users

General groups who need to track and split shared expenses.

## Solution

No authentication needed except for the sheet creator, simple easy to share URL for the sheet, simple easy to share URL where participants can join by setting a name, clear debts settlement and minimum transactions, modern UI.
